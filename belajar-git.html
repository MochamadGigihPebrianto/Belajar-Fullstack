Beberapa kode penting untuk GIT:
git add . (menambahkan semua file dari working area ke stage) git status (untuk melihat status file)
git diff (untuk melihat file mana yang barusan diubah)
git restore -S nama.js (mengembalikan file dari stage ke working area)
git restore (mengembalikan/undo perubahan pada file di working area)
git commit -m "nama file" (untuk mengubah file dari stage menjadi commit)
git log (untuk melihat history)
git reset --hard kodefile.js (untuk kembali ke commit tujuan)
git log --oneline (untuk menampilkan history secara singkat dan sederhana)
